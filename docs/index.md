<div class="feature-grid">
  <div class="feature-card">
    <p class="feature-card__title">Pure Python</p>
    <p class="feature-card__body">No Git binary required. Works anywhere Python does — Linux, macOS, Windows.</p>
  </div>
  <div class="feature-card">
    <p class="feature-card__title">Rust extensions</p>
    <p class="feature-card__body">Optional Rust extensions for performance-critical paths, automatically used when available.</p>
  </div>
  <div class="feature-card">
    <p class="feature-card__title">Porcelain API</p>
    <p class="feature-card__body">High-level <code>clone</code>, <code>pull</code>, <code>push</code>, <code>commit</code>, <code>diff</code> — mirrors Git's own commands.</p>
  </div>
  <div class="feature-card">
    <p class="feature-card__title">Low-level API</p>
    <p class="feature-card__body">Direct access to objects, refs, pack files, and the full Git data model.</p>
  </div>
  <div class="feature-card">
    <p class="feature-card__title">Protocol v2</p>
    <p class="feature-card__body">Git protocol v2, packfile-uris, bundle URI, and SHA-256 repository support.</p>
  </div>
  <div class="feature-card">
    <p class="feature-card__title">Python 3.10+</p>
    <p class="feature-card__body">Supports CPython 3.10–3.14 and PyPy. Typed throughout.</p>
  </div>
</div>
